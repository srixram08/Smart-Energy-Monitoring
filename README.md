# Smart Energy Monitoring System

## Features
- Simulates energy usage using ESP32 (Wokwi)
- Sends data to ThingSpeak cloud
- Detects occupancy using PIR
- Controls LED based on motion

## Tools Used
- Wokwi
- ThingSpeak
- ESP32
- VS Code
- Groq
- Lovable
- Claude

## Output
- Real-time graphs of power and occupancy
- Predict the Cost and power consumed
